# Product Overview

**Internal Hub** is a React-based web application for managing rooms and members with interactive features.

## Core Features

- **Room Management**: Create, edit, and manage rooms
- **Member Management**: Add, edit, and manage members within rooms
- **Wheel of Fortune**: Interactive fortune wheel feature for member selection
- **Theme Toggle**: Light/dark mode support
- **Firebase Integration**: Real-time data storage and hosting

## Target Environment

- Web application deployed on Firebase Hosting
- Real-time data synchronization using Firestore
- Responsive design for various screen sizes

## Key User Flows

1. Create and manage rooms
2. Add/edit members within rooms
3. Use fortune wheel for random member selection
4. Toggle between light and dark themes
