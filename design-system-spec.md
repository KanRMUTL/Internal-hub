# Design System Specification

## Components

### Badge

A small tag component for statuses.

- **Variants**: `default`, `secondary`, `destructive`, `outline`, `success`, `warning`.
- **Accessibility**: Standard HTML element.

### Tooltip

A contextual pop-up for icon buttons.

- **Primitives**: Radix UI Tooltip.
- **Accessibility**: Supports keyboard focus and screen reader announcements via Radix UI.

### Tabs

A content-switching component for organizing sections.

- **Primitives**: Radix UI Tabs.
- **Accessibility**: Full keyboard navigation (arrows to switch tabs), ARIA roles automatically applied.

### Popover

A flexible floating container for complex information.

- **Primitives**: Radix UI Popover.
- **Accessibility**: Focus management, keyboard navigation (ESC to close), ARIA attributes.
