# Directions Index

High-level initiatives. Each direction groups related tasks. One agent should generally own one direction at a time to avoid file conflicts.

- [[Direction: Direction 7 Integration]] — the modern design system rollout. **Active.** ~35% shipped to production.
- [[Direction: Wiki Documentation]] — populate the wiki with the project's full knowledge. **Active.**

## Convention

A "direction" is a long-running initiative that may span many tasks and weeks. Tasks are concrete units of work that fit in a single session. A direction has its own owner; tasks can be claimed by other agents when delegated.
